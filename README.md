# dot files

Configuration files on my development machine.

Getting Started
---------------

1. Clone repo.

    ```
    git clone https://github.com/r0y3/dotfiles.git
    ```

2. Create ```.vimrc``` symlink.

    ```
    ln -s ~/dotfiles/.vimrc ~/.vimrc
    ```

3. Create ```.tmux.conf``` symlink.

    ```
    ln -s ~/dotfiles/.tmux.conf ~/.tmux.conf
    ```
